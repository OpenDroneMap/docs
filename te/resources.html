అదనపు సూచనలు¶

వినియోగదారుల కోసం¶

కింది వనరులు ప్రారంభించడానికి మంచి ప్రదేశం:

  • ODM కోసం

  • WebODM కోసం README పేజీ

  • NodeODM కోసం

  • గ్రౌండ్ కంట్రోల్ పాయింట్స్ ఫార్మాట్ స్పెసిఫికేషన్

  • ఓపెన్‌డ్రోన్‌మ్యాప్: తప్పిపోయిన గైడ్

డెవలపర్ల కోసం¶

వినియోగదారు వనరులతో పాటు, ఈ క్రింది వాటిని కూడా చదవమని మేము సిఫార్సు చేస్తున్నాము:

  • WebODM డాక్యుమెంటేషన్: https://docs.webodm.org

  • NodeODM API స్పెసిఫికేషన్: https://github.com/OpenDroneMap/NodeODM/blob/master/docs/index.adoc

  • ODM పైప్‌లైన్ యొక్క అవలోకనం: http://community.opendronemap.org/t/where-can-i-find-background-information-on-the-concepts-of-odm/665/2

  • మేము మా ఫోరమ్‌లో కు అంకితమైన విభాగాన్ని ఉంచుతాము. సాఫ్ట్‌వేర్‌ను మెరుగుపరచడానికి ఉపయోగపడే మోషన్, మల్టీ-వ్యూ స్టీరియో, మెషింగ్, టెక్స్‌టరింగ్ మొదలైన వాటి నుండి నిర్మాణానికి సంబంధించిన ఆర్ట్ రీసెర్చ్ గురించి మరింత చదవడానికి ఇది ఒక విలువైన ప్రదేశం.
