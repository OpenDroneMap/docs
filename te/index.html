OpenDroneMap యొక్క డాక్యుమెంటేషన్‌కు స్వాగతం¶



There is a guide on using GitHub to contribute. Anyone is welcome to share their knowledge and improve our documentation. You can also report problems using GitHub issues.

డాక్యుమెంటేషన్ అనేక భాషలలో అందుబాటులో ఉంది. కొన్ని అనువాదాలు అసంపూర్తిగా ఉన్నాయి మరియు ఇప్పటికీ ఆంగ్లంలో ప్రదర్శించబడతాయి. దిగువ భాషలు కనీసం 20 శాతం అనువదించబడ్డాయి. ట్రాన్సిఫెక్స్ ద్వారా ఆ భాషలను (మరియు ఇతర భాషలను!) అనువదించడానికి మీరు మాకు సహాయపడగలరు. GitHub లో` ప్రారంభ గైడ్ ఉంది.
